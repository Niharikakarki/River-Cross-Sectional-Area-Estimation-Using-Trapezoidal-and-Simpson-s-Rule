# River-Cross-Sectional-Area-Estimation-Using-Trapezoidal-and-Simpson-s-Rule
Numerical estimation of river cross-sectional area using Simpson's 1/3 Rule and Trapezoidal Rule in Python.
The objective is to estimate the cross-sectional area of a river using numerical integration techniques. The project implements the Trapezoidal Rule and Simpson's 1/3 Rule on river depth measurements and compares the estimated results through error analysis.
